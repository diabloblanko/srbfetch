# srbfetch 1.3.2
Fast, C-Written, non-bloated alternative for neofetch and fastfetch. Work in progress!
## Whats new?
Added Unknown logo
## How to install
Run vars.sh script.
```
sh vars.sh
```
Compile srbfetch.c
```
clang srbfetch.c -o srbfetch
```
Copy the binary to $PATH
```
cp srbfetch /usr/bin
```
